교육공감수학(주) 통합 교육 플랫폼 프로토타입 (v2.0)

> **"전국 도서산간 및 교육 소외 계층 분원에 제한 없는 인프라 공급을 위한 확장형 화이트 라벨링 LMS 플랫폼"**

본 프로젝트는 프로그래머의 소스코드 수정이나 재배포 공수 없이, 전국 지점의 복지관·공부방·학원의 환경에 맞추어 **실시간으로 시스템 전체 명칭과 슬로건이 가변 동기화**되는 차세대 통합 학원 관리 솔루션의 기획 검증용 프로토타입입니다.

---

## 🎯 핵심 기획 의도 및 가치 체계 (대표님 브리핑 핵심 요약)

1. **브랜드 통일성 (Identity Architecture) 확보**
   - 기존 파편화되어 있던 로그인 전/후 화면 및 최고 관리자(Admin), 원장실(Manager) 페이지의 UI 디자인 톤앤매너를 단일 표준 사양으로 정밀 일원화했습니다.
2. **원격 보급 확장성 (White Labeling Solution) 증명**
   - 개발 인력이나 인프라 비용이 전무한 도서산간 소외 지역 운영자도 마우스 클릭 몇 번만으로 본인들 지점 전용 마스터 LMS 자산을 즉시 소유할 수 있도록 고안되었습니다.
3. **사용자 경험 (UX/UI) 고도화**
   - 모바일에 깨지던 구식 입력 폼 그리드를 칼같이 정렬하고, 뚝뚝 끊기던 서브 메뉴들에 부드러운 플로팅 카드 드롭다운 및 모달 윈도우 인터랙션을 심어 조작 편의성을 극대화했습니다.

## 📂 유기적 서브시스템 주소 구조 (인프라 매핑 맵)

웹 표준 디렉터리 분리 체계를 도입하여 실제 대형 서비스 수준의 완벽한 멀티 포털 주소 구조를 구현했습니다.

* **💻 일반 게이트웨이 대문**: `index.html`
  - *기능*: 사용자 인증 인트로 허브 및 실시간 지점명 커스텀 컨피그 제어판 내장.
* **🧒 학생 회원가입**: `signup.html`
  - *기능*: 모바일 맞춤형 회원정보 가입 폼 템플릿.
* **👩‍🏫 선생님 대시보드**: `teacher/teacher_main.html`
  - *기능*: 탭 기반 학생/클래스 전환, 대단원 아코디언 하위 커리큘럼 소거 엔진, 실시간 진행률 프로그레스 바.
* **📝 수학 학습 상세**: `teacher/study_detail.html`
  - *기능*: 개념 가이드, 실전 문제 캔버스, 인쇄 콘솔이 삼분할된 3-Column 디지털 레이아웃.
* **🔒 최고 관리자 로그인**: `admin/index.html`
  - *기능*: 이미지 엑스박스 결함을 차단한 웹 표준 SVG 탑재 다크모드 인증 창.
* **🖥️ 마스터 컨트롤 타워**: `admin/admin_main.html`
  - *기능*: 서버 리소스 관제 대시패널 및 트래픽 로그 모듈.
* **👑 원장실 운영 관리**: `manager/manager_main.html`
  - *기능*: ERP 인프라 그리드 테이블, 3축 서브 메뉴 드롭다운, 반투명 배경 블러 마스킹 클래스 등록 모달.
* **👨‍🏫 강사 정보 수정**: `teacher/teacher_edit.html`
  - *기능*: 깨지던 전화번호/인풋 칸을 수평 밀착 정렬한 반응형 ERP 폼 그리드.

## 🛠️ 실실간 시연(Demo) 가이드 동선

1. 메인 대문(`index.html`) 상단의 **[⚙️ 지역 센터명 커스텀 설정]** 버튼을 엽니다.
2. 보급하려는 타겟팅 지점명(예: `신안 안좌도 꿈나무 공부방`)을 입력한 뒤 적용 버튼을 누릅니다.
3. 시스템 전체 브랜딩이 일괄 자동 변신하는 효과를 확인합니다.
4. 이후 선생님 뷰, 원장실 뷰, 어드민 뷰로 각 주소를 타고 진입하더라도 상단 GNB 바의 지점 텍스트 명칭이 **완벽하게 실시간 동기화**되어 구동되는 핵심 원리를 브리핑합니다.
